# 🎵 BeatBox — Console Music Player

A lightweight, console-based music player built in C++. It runs on Windows and integrates with VLC or Windows Media Player to handle audio playback. Browse your songs and control playback right from the terminal — no GUI, no extra dependencies.

---

## Features
- 📋 **Display Playlist** — View all songs in a clean numbered list
- ▶️ **Play Song** — Pick any song by number, launches instantly
- ⏹️ **Stop Song** — Stops playback immediately
- ⏭️ **Next / Previous Song** — Navigate tracks with auto wrap-around
- 📁 **Simple Setup** — Just paste your song paths in a .txt file

---

## How to Run
1. Place `MusicPlayer.exe`, `MusicPlayer.cpp`, and `SongsPlaylist.txt` in the **same folder**
2. Open `SongsPlaylist.txt` and paste your song file paths — one per line, **no quotes**
3. Set your default media player to **VLC** or **Windows Media Player**
4. Run `MusicPlayer.exe` and enjoy!
