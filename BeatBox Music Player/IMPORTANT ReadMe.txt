Process of running the program:

----> Default Media player should be windows media player or VLC <----

Step 1. Place Musicplayer.exe + MusicPlayer.cpp + SongsPlaylist.txt to one folder (This is very IMPORTANT).
Step 2. Edit SongsPlaylist.txt copy paste your songs paths one by one in new lines in sequence without quotation marks (IMPORTANT: Remove the quotation Marks) and then save.
Step 3. Run the program (.exe file)
