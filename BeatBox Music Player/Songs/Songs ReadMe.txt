Process of running the program:

Step 1. Place all your songs and Musicplayer.exe + MusicPlayer.cpp + SongsPlaylist.txt to         one folder. 
Step 2. Create document(.txt) file named exact as SongsPlaylist.txt and edit it and copy         paste your songs paths one by one in new lines in sequence without qoutation marks.
Step 3. Run the program (.exe file)
